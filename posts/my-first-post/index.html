用rclone 挂载 a1p 版的 onedrive

成功的Rclone 配置文件

[ConfigName]
type = webdav
url = https://[domain-pre]-my.sharepoint.com/personal/[full-domain]/Documents
vendor = sharepoint
user = Your-Email
pass = Your-Password

URL获取

  • 通过网页登陆Onedrive
  • 登陆完成之后，文件管理界面复制地址栏的网址
    • 保留https://[domain-pre]-my.sharepoint.com/personal/[full-domain]/部分
    • 在后面加上/Documents
